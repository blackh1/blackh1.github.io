##XCTF_MISC

###题目一
Ditf

下载附件后发现是一个png文件，放入StegSolve查看也没有什么特别的，然后查看该文件比较大，猜测是文件里藏了别的东西，使用foremost进行分离尝试，得到了一个png图片和一个rar压缩包。

然后尝试解压压缩包，发现需要密码，经过初期的暴力尝试发现有亿点点困难，所以转回查看png文件。
经过StegSolve和WinHex多方向尝试并没有发现什么特别的，然后想到了改图像的高度，经过尝试后发现密码。

然后对该文件进行分析，查找flag，发现没找到，之后查找各种文件后缀，在查找png的时候找到了一个kiss.png的文件，追踪HTTP流后发现下面还有一串字符

猜测被base64加密过，进行解密后得到flag。

flag为flag{Oz_4nd_Hir0_lov3_For3ver}

###题目二
Banmabanma

下载附件后发现是一个压缩包，放入WinHex打开后并没有什么特殊的，然后正常解压出来一张斑马的图片，使用StegSolve和WinHex进行各种分析无果，突然瞅这个斑马特别像条形码，所以识别一下，果然是一个条形码。

最终得到flag为flag{TENSHINE}
识别二维码的网站为：https://online-barcode-reader.inliteresearch.com/

###题目三
stage1

下载附件后发现是一张图片，使用StegSolve打开，查看各个信道发现里面有一张二维码

对二维码进行分析得到一串字符，

看开头是03F30D0A，经过百度发现是.pyc文件，使用Easy Python Decompiler进行反编译，得到相应的python文件，使用文本编辑打开，复制粘贴运行代码即可得到flag

flag为AlphaLab，没有flag{}。
Easy Python Decompiler网址为https://sourceforge.net/projects/easypythondecompiler/
